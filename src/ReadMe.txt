# My Store

متجر إلكتروني مبني بـ Vue 3 و Vite و Pinia.

## وصف المشروع

هذا المشروع هو واجهة متجر إلكتروني كاملة مع تصفح المنتجات، إضافة إلى السلة، ودفع الطلبات، ولوحة إدارة بسيطة، ونظام طلبات.

## مميزات المشروع

- واجهة مستخدم بـ Vue 3 و Vite
- إدارة الحالة باستخدام Pinia
- تصميم سريع باستخدام Tailwind CSS
- صفحة رئيسية تعرض منتجات مميزة
- عرض منتجات حسب التصنيف
- صفحة تفاصيل المنتج
- إضافة المنتج إلى السلة
- زيادة / تقليل كمية المنتج في السلة
- حذف المنتج من السلة
- حفظ محتوى السلة في `localStorage`
- صفحة بحث عن المنتجات
- صفحة الدفع (Checkout)
- رسالة تأكيد طلب ناجح
- إدارة الطلبات
- صفحة الطلبات الخاصة بالمستخدم
- تتبع الطلبات باستخدام مكوّن `OrderTracker`
- صفحة الملف الشخصي للمستخدم
- صفحات تسجيل الدخول والتسجيل
- لوحة إدارة تعرض المنتجات والأوامر
- صفحة تفاصيل الطلب للمشرف
- إشعارات تفاعلية باستخدام SweetAlert2
- زر واتساب للتواصل السريع
- دعم الوضع الليلي/النهاري باستخدام `useDarkMode`
- تحسين تجربة المستخدم بعناصر واجهة مثل المكوّنات المخصصة
- تخزين بيانات الطلبات في `localStorage`

## المكونات الرئيسية

- Home.vue - الصفحة الرئيسة
- Categories.vue - عرض التصنيفات
- Category.vue - صفحة فئة المنتجات
- ShowProduct.vue - عرض تفاصيل المنتج
- Cart.vue - صفحة السلة
- Checkout.vue - صفحة الدفع
- OrderSuccess.vue - صفحة نجاح الطلب
- Orders.vue - صفحة الطلبات العامة
- MyOrders.vue - صفحة الطلبات الخاصة بالمستخدم
- Profile.vue - صفحة الملف الشخصي
- AdminProducts.vue - إدارة المنتجات
- AdminDashboard.vue - لوحة تحكم الإدارة
- OrderDetails.vue - تفاصيل طلب المشرف

## المتاجر (Stores)

- cart.js - إدارة سلة التسوق
- checkout.js - بيانات صفحة الدفع والحسابات
- orders.js - إدارة الطلبات
- toast.js - نظام الإشعارات

## التقنيات المستخدمة

- Vue 3
- Vite
- Pinia
- Vue Router
- Tailwind CSS
- SweetAlert2
- Firebase (موجود في التبعيات)
- Font Awesome

## تشغيل المشروع

1. تثبيت الحزم:

```bash
npm install
```

2. تشغيل البيئة المحلية:

```bash
npm run dev
```

3. بناء المشروع للإنتاج:

```bash
npm run build
```

4. معاينة البنية:

```bash
npm run preview
```

## ملاحظات

- قد تضاف المزيد من ميزات الإدارة والتكامل مع Firebase لاحقاً.
- يتم تخزين البيانات محلياً في المتصفح الآن باستخدام `localStorage`.
